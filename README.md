<h1> Atividade AC2 </h1>
<p>API de gerenciamento de professores em uma escola. Essa API permite o controle das informações pessoais dos professores, bem
como os dados das turmas e disciplinas ministradas por eles.
Com essa API, a escola poderá listar, atualizar e organizar as informações dos professores
e suas turmas de maneira centralizada e eficiente.</p>
